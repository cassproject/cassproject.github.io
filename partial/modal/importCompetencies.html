CSV files should be formatted so that each row contains a single competency's information. The only required column is the Competency name, other properties are optional. If you include an ID column, it can either be in the format of <URL> / <Unique ID> or just list the Unique ID.

The URL of the current server will replace a URL in the ID column, or prepend to the ID if only the unique ID. Unique ID's must only contain alpha or symbol characters.
